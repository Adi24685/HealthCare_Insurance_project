# 🏥 Healthcare Insurance Cost Prediction

This project is a Machine Learning-based web application that predicts **healthcare insurance costs** based on user input such as age, BMI, smoking habits, and more.

It uses multiple ML algorithms and provides predictions through an interactive **Streamlit web app**.

---

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🤖 Multiple Machine Learning Models:
  - Linear Regression
  - Decision Tree
  - Random Forest
- ⚙️ Hyperparameter Tuning
- 📈 Model Evaluation & Comparison
- 🌐 Interactive UI using Streamlit
- 🔮 Real-time Insurance Cost Prediction

---

## 🧠 Problem Statement

Healthcare insurance costs vary significantly depending on factors like age, BMI, smoking habits, and region. 

This project aims to:
- Analyze the dataset
- Identify key factors affecting insurance charges
- Build predictive models
- Provide accurate cost estimation for users
